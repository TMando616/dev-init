# Product Overview

プログラミング初学者向けの教育Webアプリ「DevInit」は、開発環境の構築を必要とせず、ブラウザ上で直接コードを記述・実行できる学習プラットフォームです。

## Core Capabilities

- **ブラウザ完結型の演習環境**: コードエディタと実行結果表示を同一画面で提供。
- **セキュアなコード実行**: Dockerを利用した隔離環境でのユーザーコード実行。
- **教材管理システム**: 管理者によるMarkdown形式の問題作成とユーザー管理。

## Target Use Cases

- **プログラミング学習の導入**: 環境構築のハードルを下げ、即座に学習を開始する。
- **演習問題の実施とフィードバック**: 実行結果や模範解答との比較による自己学習。

## Value Proposition

「Completion over Perfection（完璧より完了）」を掲げ、MVPとして1言語の実行環境をブラウザ上で完結させることで、学習者が最も早く「動くコード」を体験できる価値を提供します。
